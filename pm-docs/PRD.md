# Project Name: The Diamond Edit 

# Project Goal: 
A Website that gives diamond and jewelry entuasiasts regualr info about the world of high jewelry and diamond auctions through a visually appealing and easy-to-navigate website.

    -

# Core Features

1. Featured Articles Section: Showcases 2–3 short stories or highlights about jewelry trends or auction events.  
2. Gallery Showcase: Displays images of high jewelry and diamonds with short captions.  
3. Random Diamond Fact: Show a different fun fact each time the page is reloaded (JavaScript).

    -

# Website Pages

1. Home – Index page with featured content (articles about High-jewelry and current Auctions) and random diamond fact.  
2. Gallery – A simple photo grid with either showcasing jewelry or diamond images (With small facts about the brand or the origin).  
3. Auctions – List of notable upcoming and past jewelry/diamond auctions.  
4. Contact – Contact form for inquiries.

    -

# Design & Interaction featurees: 

1. Theme: Clean, luxurious, and minimal.  
  - Subtle gradient background (White and Gold) with gold and black accents.  
  - Title Font: Playfair Display.  
2. Interactivity:  
  - Random fact each time page reloaded.  
  - Hover effect for gallery images with name/description appearing on the bottom.  
3. Layout  
  - Navigation bar and footer for each page.
